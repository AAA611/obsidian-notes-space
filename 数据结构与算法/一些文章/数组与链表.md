## 复杂度对比

|      | 数组      | 链表      |
| ---- | ------- | ------- |
| 插入元素 | $O (n)$ | $O (1)$ |
| 访问元素 | $O (1)$ | $O (n)$ |
| 删除元素 | $O (n)$ | $O (1)$ |
| 查找元素 | $O (n)$ | O $(n)$ |
### 插入元素

由于数组是连续结构，所以数组在插入、删除元素时，后面所有的元素都要进行移动，所以数组的插入、删除操作的复杂度都是 $O (n)$
